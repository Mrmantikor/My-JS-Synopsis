Пошук DOM-елементів

  • element.querySelector(selector) - повертає перший елемент, який відповідає вказаному CSS селектору всередині елемента.
  • element.querySelectorAll(selector) - повертає всі елементи, які відповідають вказаному CSS селектору всередині елемента.

Властивості DOM-елемента

  • element.textContent - містить текстовий вміст елемента без HTML-тегів.
  • element.innerHTML - містить HTML-вміст елемента.
  • element.style - об'єкт, що містить вбудовані стилі елемента.

CSS класи на DOM-елементах

  • element.classList.contains(className) - перевіряє, чи є клас у елемента.
  • element.classList.add(className) - додає клас.
  • element.classList.remove(className) - видаляє клас.
  • element.classList.toggle(className) - додає або видаляє клас.
  • element.classList.replace(oldClassName, newClassName) - замінює клас.

Створення та видалення DOM-елементів

  • document.createElement(tagName) - створює новий елемент.
  • element.append(el) - додає вміст у кінець.
  • element.prepend(el) - додає вміст на початок.
  • element.remove() - видаляє елемент.
  • element.insertAdjacentHTML(position, string) - додає HTML-код у певне місце.

Події

  • element.addEventListener(event, handler) - додає обробник події.
  • element.removeEventListener(event, handler) - видаляє обробник події.
  • keydown - подія натискання клавіші.
  • submit - подія відправки форми.
  • change - подія зміни значення.
  • input - подія введення даних.
  • focus - подія отримання фокусу.
  • blur - подія втрати фокусу.
